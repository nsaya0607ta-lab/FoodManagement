# Reizo（レイゾー）

冷蔵庫在庫管理・レシピ提案アプリの MVP 実装です。設計書に基づき、以下の体験を実装しています。

- 冷蔵庫を模したホーム画面（保存場所ごとのタップ操作）
- 保存場所別の在庫一覧・食材詳細・残量調整
- 食材の手動登録（カテゴリに応じた個数／容量／重量／枚数の入力UI）
- 自動消費ルール（毎日・平日・曜日指定などの繰り返し設定）
- 在庫に基づくレシピ提案とスコアリング
- レシピ実行後の在庫更新（在庫変更は必ず履歴として保存）

レシートOCR・バーコード読み取り・画像解析などのAI/外部連携機能は、今後のバージョンで対応予定のためモック化しています。

## セットアップ

```bash
npm install
npm run dev
```

http://localhost:3000 で起動します。

初回アクセス時に `data/db.json`（gitignore対象）へデモ用の初期在庫データが自動生成されます。設定画面から「デモデータをリセット」でいつでも初期状態に戻せます。

## 技術構成

- Next.js (App Router) / TypeScript / Tailwind CSS
- データ永続化: JSON ファイル（`lib/db.ts`）によるシンプルなストア。本番運用時は PostgreSQL 等への置き換えを想定しています。
- ビジネスロジック: `lib/store.ts`（在庫の増減・履歴記録・自動消費ルールの実行・レシピの在庫一致スコアリング）

## ディレクトリ構成

```
app/        画面・APIルート（Next.js App Router）
components/ クライアントコンポーネント
lib/        型定義・ビジネスロジック・データストア
```
